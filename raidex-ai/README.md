# Raidex AI Engineering Platform

This workspace runs a local AI Engineering Operating System for Raidex development, testing, code quality, security, monitoring, memory, RAG search, and reporting.

Start here:

- `AI_SETUP.md`
- `AGENTS.md`
- `MODEL_GUIDE.md`
- `WORKFLOW.md`

Quick start:

```powershell
powershell -ExecutionPolicy Bypass -File .\raidex-ai\scripts\setup.ps1
python .\raidex-ai\scripts\raidex_ai.py daily-report
```

This is not customer-facing app code.
